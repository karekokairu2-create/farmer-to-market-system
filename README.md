# Farmer-to-Market Produce System

## Project Overview
A web-based system to connect farmers directly with customers, streamlining the distribution of avocados, mangoes, and oranges. The system reduces manual processes and improves transparency in orders, payments, and inventory.

## Objectives
1. Implement secure user authentication for Admin, Customer, Farmer, and Manager roles.
2. Provide a product listing interface for farmers to update available produce.
3. Enable customers to browse products, place orders, and track order status.
4. Integrate basic payment functionality linked to orders.
5. Provide managers with dashboards for monitoring sales and inventory.
6. Maintain clear documentation and version control through GitHub.

## System Scope
**Included:** Authentication, product listing, order placement, payment integration, dashboards.  
**Excluded (for now):** Real-time delivery tracking, SMS/email notifications, advanced analytics, mobile app development.

## Technology Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js with Express
- Database: MongoDB (in-memory for MVP, scalable later)
- Hosting: Render or GitHub Pages
- Version Control: GitHub

## Repository Structure
- `/frontend` – UI files
- `/backend` – Express server and routes
- `/docs` – Planning documents, ERD, mockups
- `/README.md` – Overview and setup instructions
